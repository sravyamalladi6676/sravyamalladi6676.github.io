SNIF: Stock Network Inference Framework

Project. Learn daily stock-return embeddings with an autoencoder, infer a dynamic graph from embedding similarities, and predict next-day movement using a GCN+LSTM student model. Emit per-stock JSON ({"stock":"AAPL","snif_prob":0.68}) and map to BUY/SELL/HOLD with a DecisionAgent. Includes a Backtrader harness.

My contribution. I implemented return ingestion and cleaning, the PyTorch autoencoder + trainer, topology inference (cosine + threshold), the GCN+LSTM model and TorchScript export, the inference pipeline that recomputes adjacency and outputs JSON, the Crew AI DecisionAgent integration (with safe fallback), and the Backtrader strategy + analyzers (Sharpe, returns, drawdown). I wrote unit/integration hooks and CLI entries for backtests.

Deeper Overview

Architecture (high level)

  • Daily returns matrix (days × stocks) → Autoencoder → latent z_t
  • Cosine similarity on z_t → threshold → Adjacency
  • GCN(adj, features) → LSTM over time → next-day class logits
  • Inference: TorchScript model + today’s adjacency → {"stock","snif_prob"}
  • DecisionAgent: map prob to BUY/SELL/HOLD (LLM optional; HOLD fallback)
  • Backtesting: signal → trade; analyzers: Returns, TimeReturn, Sharpe, DrawDown
